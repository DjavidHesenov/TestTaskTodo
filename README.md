# 📌 To-Do App Setup Guide

## 🚀 How to Run

#### 1️⃣ Clone the Repository
#### 2️⃣ Start Backend
1. `cd todo-backend`

2. `npm install`

3. `npm run start`

📌 Runs on http://localhost:5000

#### 3️⃣ Start Frontend
1. `cd ../todo-frontend`

2. `npm install`

3. `npm run dev`

📌 Runs on http://localhost:5173
